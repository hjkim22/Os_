- CPU의 구성요소 ALU, CU, 레지스터 각각의 역할
    - ALU: 산술 연산과 논리 연산을 수행.
    - CU: 명령어의 해독과 실행을 조정하여 다른 장치들을 제어.
    - 레지스터: CPU 내부에 있는 고속 메모리, 데이터나 주소 등을 일시적으로 저장하거나 조작.


- 메인 메모리와 보조기억장치의 차이
    - 메인 메모리는 CPU에 직접 접근할 수 있는 주 기억장치, 프로그램과 데이터를 일시적으로 저장
    - 보조기억장치는 주로 하드 디스크와 같은 비휘발성 기억장치를 가리키며, 메인 메모리보다 접근 속도가 느리고 용량이 큼.
        보조기억장치는 메인 메모리의 확장으로 사용되며, 프로그램 및 데이터의 영구적인 보존과 추가 저장 공간을 제공


- 버스 시스템 데이터의 종류 세 가지로 구분
    - 데이터 버스: 데이터 전송을 담당하는 버스, CPU, 메모리, 입출력 장치 간의 데이터 전송에 사용.
    - 주소 버스: 메모리나 입출력 장치에 데이터의 위치를 지정하기 위한 버스, 메모리 주소 또는 입출력 장치의 주소를 전송.
    - 제어 버스: 제어 신호를 전송하기 위한 버스, 명령어의 실행이나 입출력 동작을 제어하는 데 사용.
